Well, you're probably wondering why the site looks so bare.

If you're coming here from the wonderful Responsive Design Weekly, or maybe just on your own, I'm embarrassed to admit my site is a bit of a mess at the moment. Wanting to try something completely new and needing to feed a powerful hunger for git and markdown, I decided to start completely afresh on my site and go with Jekyll hosted on Github Pages. As Justin highlighted in the newsletter, I have been working on the design aspect of this update for a while now, but haven't had the time or the kick up the ass to get to it and get it done. What better motivation than shiny, new technology? ... So anyway, basically I migrated my domain over to Github with the barebones of a template and absolutely no content whatsoever, so please bear with whatever state you are reading this in. :)
